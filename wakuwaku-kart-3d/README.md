# わくわくカート 3D

マリオカート64風の、4歳向け3Dカートレースゲーム。iPhone / iPad の縦画面・横画面どちらでも遊べます。

## あそびかた

1. `index.html` をブラウザで開く(ローカルサーバー推奨: `python3 -m http.server`)
2. 「すたーと!」をタップ
3. すきなキャラクターとコースをえらぶ
4. アクセルは自動! 画面の **ひだり半分をタッチ=ひだりに曲がる**、**みぎ半分をタッチ=みぎに曲がる**
5. 3しゅう はしってゴールしよう!

PCの場合は矢印キー(←→)でも曲がれます。

## 4歳向けの工夫

- アクセル自動・ハンドルだけの超シンプル操作
- 道なりに進む強力な走行アシスト+コース外に出ない優しい壁
- 敵カートはラバーバンドAIで、がんばれば勝てる難易度
- 失敗要素なし: アイテムは全部うれしいもの(ダッシュ・スター・コイン・にじパワー)
- ひらがな中心の大きなUI・大きなボタン

## たのしい要素

- キャラクター6人(うさぎ・ねこ・ぱんだ・かえる・ひよこ・こあら)
- コース3種類(にじいろサーキット・うみべビーチ・おかしのくに)
- コイン・アイテムボックス・ダッシュ床・わって遊べるふうせん
- 虹のアーチ・熱気球・にこにこ太陽・ゆれる海などの賑やかな景観
- ゴールで花火と紙吹雪のおいわい+表彰台

## 技術

- Three.js r160(`vendor/` に同梱、ビルド不要の素のES Modules)
- 効果音・BGMはWebAudioでリアルタイム合成(外部アセットなし)
- 道路はCatmull-Romスプラインから動的生成、テクスチャはCanvasで生成

## 構成

```
index.html
css/style.css
js/
  main.js            エントリーポイント・シーン遷移
  core/              エンジン・入力・音・HUD・ユーティリティ
  game/              コース・カート・走行・アイテム・エフェクト・景観・データ
  scenes/            タイトル・レース・ゴールの各シーン
vendor/three.module.min.js
```
