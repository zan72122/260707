# にじいろ かきごおり ゴルトン 🍧🌈

4歳児向けの、ゴルトンボード(Galton Board)× かき氷ごっこ遊びゲームです。
iPhone / iPad の縦画面・横画面のどちらでも遊べます。

## あそびかた

1. **タップしてはじめる** — タイトル画面をタップ。
2. **ゆびで タッチ&ドラッグ** — にじいろの雲がゆびを追いかけて、シロップの粒をぽとぽと落とします。
3. 粒は氷の結晶ピンに当たって**左右にころころ分かれ**、かき氷の上に**カラフルな山**になって積もります(ゴルトンボードの物理)。
4. 下のボトルで**シロップの色**を選べます(いちご・みかん・レモン・メロン・ソーダ・ぶどう・にじいろ)。
5. たまに**星・ハート・さくらんぼ・あめちゃん**の特別トッピングが落ちてきます。
6. 55粒ごとに**レインボータイム**! 空から虹色の粒が降りそそぎます。
7. かき氷がいっぱいになったら「**いただきます!**」ボタンが登場。
8. しろくまがスプーンで**ぱくぱく**食べて、「ごちそうさまでした!」→「もういっかい!」でループ。

## 楽しい要素

- ピンに当たるとキラキラ光って、ペンタトニック音階の鈴が鳴る
- 粒の残像トレイル、着地のしぶき、紙吹雪、浮かぶ掛け声
- オルゴール風のやさしいBGM(WebAudioで合成、外部アセット不要)
- 応援してくれるしろくまのマスコット
- にこにこ太陽・流れる雲・お祭りガーランドの夏の背景

## 実行方法

ビルド不要の静的サイトです。このフォルダをHTTPサーバで配信してください。

```bash
cd nijiiro-kakigori-galton
python3 -m http.server 8000
# → http://localhost:8000 をSafari/Chromeで開く
```

※ ES Modules を使用しているため、`file://` 直接ではなくHTTPサーバ経由で開いてください。

## 構成

```
nijiiro-kakigori-galton/
├── index.html
├── css/style.css
└── js/
    ├── main.js              # エントリポイント
    ├── core/                # エンジン・音・パーティクル・ユーティリティ
    │   ├── engine.js        # キャンバス・ループ・ポインタ入力
    │   ├── audio.js         # WebAudio合成の効果音とBGM
    │   ├── fx.js            # きらきら・紙吹雪・浮かぶ文字
    │   └── utils.js
    ├── game/                # ゲームロジック
    │   ├── layout.js        # 縦横レスポンシブレイアウト
    │   ├── board.js         # ゴルトンボード物理(ピン衝突・左右分岐)
    │   ├── pile.js          # 山への蓄積(砂山くずれ・ヒストグラム状)
    │   └── palette.js       # シロップ色・特別トッピング定義
    ├── render/              # 描画
    │   ├── background.js    # 空・太陽・雲・ガーランド
    │   ├── bowl.js          # ガラスの器・氷の山・シロップ雲
    │   ├── mascot.js        # しろくまマスコット
    │   └── ui.js            # ボタン・メーター
    └── scenes/
        ├── title.js         # タイトル画面
        └── play.js          # プレイ~いただきます~ごちそうさま
```

## 技術メモ

- 依存ライブラリなし(Vanilla JS + Canvas 2D + WebAudio)
- 積もった粒はオフスクリーンキャンバスに焼き込み、粒数が増えても描画コストが一定
- 画面回転・リサイズ時は積もった粒の山を再構築して形を保持
- iOSのダブルタップ拡大・スクロールを抑止し、`viewport-fit=cover` でノッチ対応
