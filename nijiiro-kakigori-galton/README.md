# じゃらじゃらGO! にじいろかきごおりマシン 🍧🌈

4歳児向けの、ゴルトンボード(Galton Board)× かき氷ごっこ遊びゲームです。
お祭りのかき氷マシン(屋台)に大量の玉をため、レバーを引いて一気に流します。
iPhone / iPad の縦画面・横画面のどちらでも遊べます。

## あそびかた

1. **タップしてはじめる** — タイトル画面をタップ。
2. **シロップをためる** — 下のボトルで色を選び、ガラスタンクを**長押し**すると、その色の玉がドバドバ注がれる。色を替えながら注ぐと、あとで山に**色の地層**ができる(注いだ順序が保たれるFIFO)。
3. **レバーでGO!** — 右の赤い大レバーを**下に引っぱる**とゲートが開く。ちょっと引く=ちょろちょろ、全部引く=どばーっ(ガチャン!)。
4. 玉は狭い漏斗で**押し合いへし合い**しながら流れ、密な氷結晶ピンの森で左右50%に分岐し、下の**仕切りスロット**にカラフルな柱として積もる → **正規分布の山**が現れる。
5. 流れ終わると**なぞり光**が稜線をスーッとなぞり、柱がぷるんと弾んで「おやまの かたち!」。
6. いっぱいになったら「**いただきます!**」→ スプーンでぱくぱく → 「ごちそうさまでした!」→「もういっかい!」。

## 楽しいギミック

- **きらきらボール**: たまに金色の玉が混ざっていて、入ったスロットがピカーッと光る
- **ピンいたずら**: 放出前にピンをタップすると「ぷにっ」と外せる(最大6本)→ 分布が偏るのを体験できる
- **巨大いちご玉**: たまにゴロンと落ちてきて通路に詰まる → タップで「ぱーん!」と割ると小玉が飛び散る
- **うちわボタン(💨)**: 横風が吹いて、流れ中の玉が片側に寄る → 分布が歪むのを体験
- **じゃらじゃら音**: 流れている玉の量に応じて粒立ちノイズが濃くなる。ピンは鈴、スロット着地は木琴(すべてWebAudio合成)

## 実行方法

ビルド不要の静的サイトです。このフォルダをHTTPサーバで配信してください。

```bash
cd nijiiro-kakigori-galton
python3 -m http.server 8000
# → http://localhost:8000 をSafari/Chromeで開く
```

※ ES Modules を使用しているため、`file://` 直接ではなくHTTPサーバ経由で開いてください。

## 技術メモ

- 依存ライブラリなし(Vanilla JS + Canvas 2D + WebAudio)
- 玉同士の衝突は空間ハッシュで近傍のみ判定(1杯あたり350〜500個、同時アクティブ最大240)
- fpsが落ちる端末では同時アクティブ数を自動で削減(240→170→110)
- 着地した玉・タンク内の玉はオフスクリーンキャンバスに焼き込み、描画コストを一定に保つ
- 画面回転・リサイズ時は飛行中の玉をタンクへ戻し、積もった柱は再構築して形を保持
- iOSのダブルタップ拡大・スクロール抑止、`viewport-fit=cover` でノッチ対応
