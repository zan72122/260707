# にじいろテニス 🎾🌈

マリオテニス64風の3Dテニスゲーム。4歳の子どもが iPhone / iPad の縦画面・横画面で遊ぶことを想定しています。

## あそびかた

1. `index.html` をブラウザで開く(または静的サーバーで配信)
2. ▶️ をタップ → すきなキャラをタップ → ✔️ で決定
3. あそびを選ぶ
   - 🎾 **ラリーたいせん** — CPUと3ポイント先取のミニマッチ(約1分)
   - 🎯 **まとあて** — まとをタップしてボールをぶつける
   - 🎈 **ふうせんわり** — ふわふわ動くふうせんをパン!
4. ラリーたいせんは操作を選べる
   - 👆 **タイミングモード** — キャラは自動で動く。ボールが来たらタップでスイング!
   - 👣 **いどうモード** — タップした場所へキャラが移動。スイングは自動!

## とくちょう

- 文字ほぼなし。アイコン・音・演出だけで遊べる
- ラリーが10回続くとボールが 🌈 レインボーに進化
- コートに ⭐ が出たらタップ(または近づく)でアイテムゲット(巨大ボール/フィーバー)
- 観客が跳ねて旗を振って応援
- 負けても「おしい!💪😊」のポジティブ演出
- BGM・効果音・ボイス風の声は全て Web Audio でリアルタイム合成(外部素材なし)
- three.js + UnrealBloom によるブルーム・ソフトシャドウ・パーティクル演出
- FPS を監視して低速端末では自動で画質を調整

## 技術構成

- three.js 0.160(`vendor/` に同梱、ビルド不要・完全オフライン動作)
- ES Modules + importmap。フレームワーク・バンドラなし
- `js/core` エンジン/入力/音/パーティクル、`js/world` コート/空/観客/キャラ、
  `js/game` ラリー/ミニゲーム、`js/screens` タイトル/HUD

## ローカル実行

```bash
cd nijiiro-tennis
python3 -m http.server 8080
# → http://localhost:8080
```
