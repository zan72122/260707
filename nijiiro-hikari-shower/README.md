# にじいろ ひかりシャワー

4歳児向けの「ひかりあそび」ゲームです。
白い光をプリズムに通すと にじ色の光が出て、部屋のいろいろなものに当てると
色がついたり、光ったり、咲いたりします。スコアはなく、報酬は「発見」です。

iPhone / iPad の縦画面・横画面のどちらでも遊べます(Safari推奨)。

## 遊び方

- **ひかり(おひさま等)** を指でドラッグ → 光の向きが変わる
- **プリズムのまんなか** をドラッグ → プリズムが動く
- **プリズムのまわり** を指でくるくる → にじの角度が変わる(勢いで回り続ける!)
- **にじの帯をつかむ** → ぐいっと曲げられる+なでると音が鳴る「にじのハープ」
- **ものを長押し** → るんるん踊りだす
- **ひかりをぶんぶん振る** → いろがまざる「おまつりモード」
- にじ色の帯を **花・コップ・アヒル・ドレス** などに当てると変化が起きる
- 新しい変化を見つけると ⭐(発見スター)がもらえる(各シーン11こ)

## おもちゃ箱ツールバー(画面下)

押すたびに切り替わるボタンが7つ並んでいます。

| ボタン | 切り替わるもの |
| --- | --- |
| いろ | 光の色: にじ → あか → きいろ → あお(単色はプリズムでも分かれない!) |
| ふとさ | 光の帯: ほそい / ふつう / ふとい |
| かたち | 光の中を流れる形: まっすぐ / ハート / ほし |
| ひかり | 光源: おひさま / でんとう / ろうそく(ゆらゆら) / おつきさま(よるモード) |
| プリズム | さんかく / ビーだま(まるい虹) / ダイヤ(4方向) / ハート / ほし / われた(ぐにゃぐにゃ) |
| おおきさ | プリズムの大きさ: ちいさい / ふつう / おおきい |
| かがみ | 鏡を置く(3枚まで)。4回目でぜんぶかたづけ |

## シーン

| シーン | あそべること |
| --- | --- |
| あさの まどべ | カーテン開閉・コップの水のゆらゆら・スプーン反射・白い花の色染め |
| おふろの にじあわ | シャボン玉・湯気の虹・手鏡の反射・アヒルぴかー・天井のゆらゆら |
| にじいろ ドレスこうぼう | 白いドレスを光で染める・しましま模様・リボン・宝石・扇風機の風 |
| あめあがりの にわ | 水たまりの虹・しずくのきらめき・つぼみが咲く・霧吹きミニ虹・てんとうむし |

## 起動方法

ビルド不要の静的サイトです。ES Modules を使うためHTTPサーバー経由で開いてください。

```bash
cd nijiiro-hikari-shower
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 技術メモ

- 依存ライブラリなし(HTML + CSS + Vanilla JS / Canvas 2D)
- 効果音は WebAudio でリアルタイム合成(音声ファイル不要・全てやわらかい音色)
- 発見スターは `localStorage` に保存され、次回も引き継がれます
- マルチタッチ対応(光とプリズムを同時に動かせる)

## フォルダ構成

```
nijiiro-hikari-shower/
├── index.html
├── css/style.css
└── js/
    ├── main.js            エントリーポイント
    ├── core/              エンジン・入力・音・パーティクル・HUD・ツールバー・ヒント・発見記録
    ├── light/             光線計算(rays)と状態(lightrig)、帯/プリズム/光源の描画
    └── scenes/            メニュー + 4つのあそびシーン + 共通あそび(scene_extras) + 小物
```
