# コロコロ★ボールランド

スーパーマリオボール(GBA)風のピンボールゲーム。4歳の子が iPhone / iPad で遊ぶことを想定しています。縦画面・横画面のどちらにも対応します。

## あそびかた

- タイトル画面をタップするとスタート。
- ボールが右のレーンにいるとき、画面をながおしすると バネがぎゅーっと縮み、はなすとボールが発射されます。
- 画面の **左半分をタッチ → 左フリッパー**、**右半分をタッチ → 右フリッパー** が動きます(マルチタッチ対応)。
- キーボードでも遊べます(← / → でフリッパー、スペース長押し→はなすで発射)。

## たのしい要素

- キノコバンパー(にこにこ顔・ぷにぷに跳ねる)
- くるくる回るコイン(取っても少し待つと復活)
- お花のスリングショット(強く弾く)
- 歩き回る「プニ」(ボールでぽよんと踏むと高得点)
- たからばこ(3回たたくとスターが飛び出す)
- スターを取ると **にじ色フィーバー**(得点2倍・画面がにじ色に)
- 500点ごとに紙吹雪とファンファーレ
- ボールを落としても「どんまい!」ですぐ次のボール(ゲームオーバーなし)

## 技術メモ

- 依存ライブラリなし(Vanilla JS + Canvas 2D + WebAudio)。`index.html` を開くだけで動きます。
- 物理は独自実装(円 vs 線分/円/回転フリッパーのカプセル、サブステップ積分)。
- 効果音・BGM はすべて WebAudio による合成音(外部アセット不要)。
- テーブルは論理座標 600x1000 で設計し、画面サイズに合わせて等倍スケール(縦横対応)。
