# そめそめ けんびきょう 🔬

4歳児が iPhone / iPad の縦画面・横画面で遊べる、**HE染色（ヘマトキシリン・エオジン染色）の工程あそび**ゲームです。透明なスライドを液につけて、小さな世界を色で見えるようにします。

## あそびかた

しずくの妖精「シズク」が指さしと光るハイライトで、次に触る場所を教えてくれます。文字が読めなくても、アイコンと音、妖精のジェスチャーで直感的に進められます。

1. 🔵 **あおい えきに ちゃぽん**（ヘマトキシリン）— 長押しゲージで核を青く染める
2. 💧 **おみずで じゃぶじゃぶ**（分別）— 左右にゆらして余分な色を流す
3. ✨ **まほうの あおい みず**（ブルーイング）— 浸け置きで赤紫 → 青へ変身
4. 🩷 **ピンクの えきに ちゃぽん**（エオジン）— 長押しゲージで細胞質をピンクに
5. 🫧 **アルコールで さっと**（脱水）— ぐるぐる混ぜる
6. 🟦 **とうめいえきと カバーガラス** — ぽんとのせる
7. 🔬 **けんびきょうで のぞく** — ピント合わせ → 倍率アップ → 「見えた！」

## 特徴

- **工程中も楽しい見た目**: 液に入れると色がじわーっと入り、ゆらすと余分な色が流れ、ブルーイングで青に変身。
- **きっちり再現**: 染め時間・分別・ブルーイング・脱水の量で、核の濃淡や色相、背景の青み、ピンクの濃さ、濁りが変化。仕上がりは顕微鏡ビューで本物の色味（核＝青紫、細胞質＝ピンク）として見える。
- **失敗なし**: どんな仕上がりでも星1〜3と優しい「味わいコメント」で褒める。
- **毎回ちがう標本**: ひふ・はっぱ・おはな・おなかのなか・ちのなか。ときどき隠れキャラ細胞（顔つき）も。
- **縦横両対応**: 画面の向きに合わせて自動でレイアウトが切り替わる。
- **賑やかな演出**: 効果音・オルゴール風BGM・褒め声風ジングル・キラキラのごほうび。

## 技術

- [Phaser 3](https://phaser.io/)（`vendor/phaser.min.js` に同梱、オフラインでも動作）
- グラフィックはすべてコード描画（Canvas/Graphics）、音は Web Audio API による合成音。外部アセットなしで軽量。

## 起動

静的ファイルなので、このフォルダを HTTP サーバーで配信するだけで動きます。

```bash
cd somesome-kenbikyo
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
